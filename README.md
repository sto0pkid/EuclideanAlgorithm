Euclid's famous algorithm to compute the greatest common divisor of two natural numbers.

Build:

> g++ euclid.cpp -o euclid

Run:

> ./euclid 1234 123456
2


> ./euclid -v 51 68

or:

> ./euclid --verbose 51 68
68 = 51*1 + 17
51 = 17*3 + 0

gcd(51, 68): 17

Testing...

51/17 = 3
68/17 = 4

In loving memory of Euclid.
